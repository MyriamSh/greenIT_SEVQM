À l'ère du Green IT : Maîtrisez les codes !

Le numérique, comme toute industrie, a son rôle à jouer dans la préservation de l’environnement.

Comment, en tant que développeurs ou autres professionnels du web, pouvons-nous changer nos pratiques afin de réduire notre empreinte écologique ?

Les impacts environnementaux du numérique en France

Comment rendre nos pratiques plus respectueuses ?

Action 1 : Les équipements

Adopter une consommation responsable d’équipement, au moyen d'habitudes comme la réparation, le recyclage ou un entretien régulier…

  • Durée de vie des appareils
  • Utilisation dans l'entreprise
  • Responsabilisation des entreprises
  • Utiliser des énergies green
  • Recyclage des composants

Action 2 : L'EcoConception

Concevoir des services plus économes en ressources numériques sur la base de méthodes et bonnes pratiques.

Développement

  • Favoriser l'approche mobile first
  • Mutualiser les blocs de pages
  • Favoriser un design épuré et sobre
  • Éliminer les fonctionnalités non essentielles
  • Favoriser les pages statiques
  • Grouper les déclarations css similaires h1, h2, h3

Hébergement

  • Stocker localement les données statiques
  • Limiter le nombre de CSS
  • Utiliser un hébergeur « green »
  • Stocker les données/fichiers dans le cloud
  • Eviter les redirections
  • Utiliser les serveurs virtuels

Design

  • Limiter le nombre d'images et vidéos
  • Maximiser l'utilisation de pictogrammes
  • Limiter le nombre de typographies
  • Fournir une css print la plus dépouillée possible
  • Limiter le nombre de requêtes http
  • Limiter les “sliders”

Vous avez maintenant toutes les clés en main pour rendre votre site internet green !
